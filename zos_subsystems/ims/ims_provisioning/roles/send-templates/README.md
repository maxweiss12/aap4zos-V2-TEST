Send templates
=========

This role is a utility used by other roles to send the entire directory of template files to the z/OS system.

Requirements
------------

* None

Role Variables
--------------

- ### ** path **
   The path of the directory to send.


Dependencies
------------

None

Example Playbook
----------------

```yaml
  - include_role: 
      name: send-templates
        path: <local directory containing files to send>

```
